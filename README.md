# Personal Portfolio Website

A modern, responsive portfolio website built with HTML5, Tailwind CSS, and vanilla JavaScript.

## Features

- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- **Modern UI**: Clean, professional design with smooth animations and transitions
- **Dark Theme**: Elegant dark color scheme with green accent colors
- **Interactive Elements**: 
  - Animated navigation with hover effects
  - Rotating profile border animation
  - Smooth scrolling between sections
  - Mobile-responsive hamburger menu
- **Social Media Integration**: Links to your social profiles
- **Statistics Section**: Display your achievements and metrics

## Sections

1. **Hero Section**: Introduction with profile image and key information
2. **Resume**: Your professional resume (placeholder - ready for content)
3. **Certificates**: Your certifications and achievements (placeholder)
4. **Work**: Portfolio of your projects (placeholder)
5. **Contact**: Contact information and form (placeholder)

## Customization

### Personal Information
Edit the following in `index.html`:
- Replace "Your Name" with your actual name
- Update the description paragraph with your personal bio
- Add your social media links
- Replace the placeholder profile image with your photo
- Update the statistics with your actual numbers

### Colors
The theme uses a green accent color (`#10b981`). You can customize this by:
- Changing the CSS variables in the `<style>` section
- Updating Tailwind color classes (e.g., `text-green-400`, `bg-green-500`)

### Adding Content
Each section is ready for you to add your content:
- **Resume**: Add your work experience, education, and skills
- **Certificates**: Display your certifications with images and descriptions
- **Work**: Showcase your projects with images, descriptions, and links
- **Contact**: Add a contact form or your contact information

## Technologies Used

- **HTML5**: Semantic markup
- **Tailwind CSS**: Utility-first CSS framework
- **Font Awesome**: Social media icons
- **Vanilla JavaScript**: Interactive features and smooth scrolling

## Getting Started

1. Open `index.html` in your web browser
2. Customize the content with your personal information
3. Add your profile photo (replace the placeholder image)
4. Update social media links
5. Add content to the remaining sections

## Browser Support

This website is compatible with all modern browsers:
- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)
